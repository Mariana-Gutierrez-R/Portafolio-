## 🚀 Sobre el Proyecto

Este repositorio contiene mi portafolio personal, diseñado y desarrollado desde cero para ofrecer una experiencia de usuario (UX) inmersiva y premium. Su propósito es presentar de manera elegante y profesional mis habilidades técnicas, mi formación académica y los proyectos en los que he trabajado como estudiante de Ingeniería Informática.

## ✨ Características Principales

*   **Diseño Premium (Modo Claro):** Interfaz limpia y moderna basada en una paleta de colores claros con acentos verde esmeralda y degradados, implementando efectos visuales avanzados como *Glassmorphism*.
*   **Interactividad y Animaciones:**
    *   Efecto interactivo *Spotlight* en el cursor con tonalidades verdes.
    *   Animaciones *Scroll Reveal* (Intersection Observer) para una carga de contenido progresiva y elegante.
    *   Micro-interacciones en botones y tarjetas de habilidades que mejoran notablemente la retención.
*   **Arquitectura Responsiva:** Diseñado usando una filosofía *Mobile-First* gracias al sistema de grillas ágil de Bootstrap 5.
*   **Tipografía Moderna:** Uso de fuentes optimizadas de Google Fonts (`Outfit` e `Inter`) para garantizar excelente legibilidad, modernidad y jerarquía visual.

## 🛠️ Stack Tecnológico

## 🌐 Instrucciones de Despliegue Local

Si deseas ejecutar este proyecto en tu entorno local para visualizarlo o modificarlo:

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/Mariana-Gutierrez-R/tu-repositorio.git
    ```
2.  Navega al directorio principal:
    ```bash
    cd tu-repositorio
    ```
3.  Inicia un servidor web local. Por ejemplo, si tienes Python instalado:
    ```bash
    python -m http.server 3000
    ```
4.  Abre tu navegador favorito y accede a `http://localhost:3000`
